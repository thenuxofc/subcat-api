📝 SubtitleCat API

Unofficial API for subtitlecat.com - Search and download subtitles programmatically

GET Search
/api/search?q={query}

Search for subtitles by keyword

Example: 
https://your-domain.netlify.app/api/search?q=Iron+man
GET Details
/api/subtitle-details?path={subtitle_path}

Get detailed information about a specific subtitle including available languages and original content

Example: 
https://your-domain.netlify.app/api/subtitle-details?path=/subs/589/Iron%20Man-Armored%20Adventures%20%282009%29%20-%20S01E14%20-%20Man%20and%20Iron%20Man.ro.html
GET Download
/api/subtitle-download?path={download_path}

Download subtitle file directly

Example: 
https://your-domain.netlify.app/api/subtitle-download?path=/subs/770/Iron%20Man-Armored%20Adventures%20(2009)%20-%20S01E14%20-%20Man%20and%20Iron%20Man.ro-en.srt

🔧 Usage Examples

JavaScript (Fetch API)

// Search for subtitles
const searchResponse = await fetch('https://your-domain.netlify.app/api/search?q=Iron+man');
const searchData = await searchResponse.json();
console.log(searchData.results.items);

// Get subtitle details
const detailsResponse = await fetch(`https://your-domain.netlify.app/api/subtitle-details?path=${encodeURIComponent(searchData.results.items[0].relative_url)}`);
const details = await detailsResponse.json();
console.log(details.available_languages);

// Download subtitle
const downloadUrl = details.available_languages.find(lang => lang.name === 'English').download_url;
window.open(`https://your-domain.netlify.app/api/subtitle-download?path=${encodeURIComponent(downloadUrl)}`);

Python

import requests

# Search
response = requests.get('https://your-domain.netlify.app/api/search?q=Iron+man')
data = response.json()

# Get first subtitle details
subtitle_path = data['results']['items'][0]['relative_url']
details = requests.get(f'https://your-domain.netlify.app/api/subtitle-details?path={subtitle_path}').json()

# Download English subtitle
english_sub = next(lang for lang in details['available_languages'] if lang['name'] == 'English')
subtitle_content = requests.get(f'https://your-domain.netlify.app/api/subtitle-download?path={english_sub["download_url"]}').text
print(subtitle_content[:500])  # Print first 500 chars

cURL

# Search
curl "https://your-domain.netlify.app/api/search?q=Iron+man"

# Get details
curl "https://your-domain.netlify.app/api/subtitle-details?path=/subs/589/Iron%20Man-Armored%20Adventures%20%282009%29%20-%20S01E14%20-%20Man%20and%20Iron%20Man.ro.html"

# Download
curl -O "https://your-domain.netlify.app/api/subtitle-download?path=/subs/770/Iron%20Man-Armored%20Adventures%20(2009)%20-%20S01E14%20-%20Man%20and%20Iron%20Man.ro-en.srt"

🚀 Deployment to Netlify

  1. Push this code to a GitHub repository
  2. Login to Netlify
  3. Click "New site from Git"
  4. Select your repository
  5. Build settings:
    • Build command: npm install
    • Publish directory: public
  6. Click "Deploy site"

⚠️ Important Notes

  • This is an unofficial API - use responsibly
  • Respect the website's terms of service
  • Consider implementing rate limiting for production use
  • Some subtitle files may require translation (marked as 'translate_required')
